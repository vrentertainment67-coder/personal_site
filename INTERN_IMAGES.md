# Intern Task — Guest Photos Only

Each episode page supports an optional guest photo.
Your only job is to source or create a photo for each guest.

## Where to add photos
1. Save the image to: `public/images/vicfix/guests/`
2. Filename format: `[slug].jpg` (e.g., `dj-whosane.jpg`, `carlton-braganza.jpg`)
3. Open the corresponding `.md` file in `src/content/vicfix/`
4. Update the `guestPhoto` field:
   ```
   guestPhoto: "/images/vicfix/guests/dj-whosane.jpg"
   ```
5. Save. The page updates automatically on next deploy.

## Image specs
- Square or portrait format preferred (1:1 or 3:4 ratio)
- Minimum 600×600px
- Dark or neutral background works best with site aesthetic
- Do NOT use blurry, watermarked, or low-quality images

## Photo sources (try in this order)
1. The episode thumbnail on YouTube (right-click → save)
2. Guest's Instagram (@handle) — use a clear profile or performance shot
3. Google image search for their name + professional photo
4. Screenshot from the episode itself (pause on a good moment)
